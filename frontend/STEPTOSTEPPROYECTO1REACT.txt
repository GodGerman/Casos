STEP TO STEP PROYECTO1 REACT+WEBPACK+BABEL

1.‐ INSTALAR REACT: https://nodejs.org/en/
2.‐DESDE LA LINEA DE COMANDOS (cmd) TECLAR node –v, REVISAR QUE SE TENGA INSTALADO
NODEJS
3.‐CREAR LA CARPETA componentes
4.‐ABRIR UNA VENTANA DE NODE.JS Y METERSE HASTA LA RUTA DE LA CARPETA DE componentes
5.‐TECLAR EL COMANDO npm init –y PARA INICIALIZAR EL ARCHIVO DE CONFIGURACION JSON, SIN
HACER NINGUNA PREGUNTA
6.‐INSTALACION DE WEBPACK E INTERFAZ DE LINEA DE COMANDOS (cli) TECLEANDO EL SIGUIENTE
COMANDO npm install webpack webpack‐cli ‐‐save‐dev ESTABLECIENDO EL MODO DE
DESARROLLO, CREANDO LA NODE_MODULES, Y AGREGANDO LAS DEPENDENCIAS DEL WEBPACK Y
EL CLI AL ARCHIVO DE CONFIGURACION JSON
7.‐ INSTALAR BABEL PRESETS: npm i @babel/core babel‐loader @babel/preset‐env
@babel/preset‐react ‐‐save‐dev
// babel preset env SE UTILIZA PARA BAJAR LA VERSION DE ECMASCRIPT 6 A LA VERSION 5
// babel preset react PARA LA TRANSFORMACION JSX
8.‐CREAR ARCHIVO: .babelrc (SE ANEXA ARCHIVO) EN LA RAIZ DEL PROYECTO
9.‐COMANDO DE EMPAQUETAMIENTO, ABRIR EL ARCHIVO package.json, Y EN EL CODIGO:
"scripts": {
"test": "echo \"Error: no test specified\" && exit 1"
}
SUSTITUIRLO POR:
"scripts": {
"build": "webpack"
}
CON ESTO LLAMAREMOS AL PROYECTO DE WEBPACK POR MEDIO DEL COMANDO:
npm run build
10.‐CREAR EL ARCHIVO: webpack.config.js (SE ANEXA ARCHIVO) EN LA RAIZ DEL PROYECTO
11.‐CREAR LA CARPETA src DENTRO DEL PROYECTO, Y AGREGAR LOS ARCHIVOS: Aplicacion.jsx,
Hola.jsx (SE ANEXAN LOS ARCHIVOS)
12.‐ INSTALAR REACT y REACT‐DOM: npm i react react‐dom
13.‐CREAR EL ARCHIVO index.html (SE ANEXA ARCHIVO) EN LA RAIZ DEL PROYECTO
14.‐CONSTRUYE EL PROYECTO CON: npm run build
15.‐EJECUTA LA PAGINA index.html (SE ANEXA ARCHIVO) QUE DEBE DE IR EN LA RAIZ DEL
PROYECTO
